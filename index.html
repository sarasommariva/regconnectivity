    Regconnectivity¶

    Welcome to the documentation of regconnectivity, a set of codes for simulating realistic brain activity and the corresponding MEG sensor level recordings, and for computing the optimal regularisation parameters for the Minimum Norm Estimate (MNE) of brain activity and for the susequent connectivity estimation.

    A detailed description of the simulation pipeline can be found in the original paper [1].

    Download¶

    All codes can be freely downloaded from the github page.
